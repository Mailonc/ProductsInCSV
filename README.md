# File Manipulation Java 8 and Java 21

Projeto desenvolvido para praticar manipulação de arquivos utilizando Java 8 e Java 21. O sistema realiza leitura e escrita de arquivos CSV, criação automática de diretórios e processamento de dados de produtos.

## Funcionalidades

- Leitura de arquivos CSV
- Escrita de arquivos CSV
- Criação automática de pastas
- Processamento de dados
- Cálculo do valor total dos produtos
- Uso de `BufferedReader` e `BufferedWriter`
- Uso moderno de `Files` e `Path` no Java 21

## Tecnologias Utilizadas

- Java 8
- Java 21
- API IO
- API NIO.2
- Git
- GitHub
- Linux Ubuntu

## Estrutura do Projeto

O programa:

1. Lê um arquivo `source.csv`
2. Processa os dados dos produtos
3. Calcula o valor total (`preço * quantidade`)
4. Gera automaticamente o arquivo `summary.csv` na pasta `out`

## Exemplo de Entrada

```csv
TV,1000.00,2
Mouse,40.00,3
Tablet,400.00,1
```

## Exemplo de Saída

```csv
TV,2000.00
Mouse,120.00
Tablet,400.00
```
